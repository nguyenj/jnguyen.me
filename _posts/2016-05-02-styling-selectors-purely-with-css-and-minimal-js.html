There are many jQuery Plugins in the wild to help stylized the select form tag including the dropdown UI. This example will provide a path to stylized the select form tag, not the dropdown. We will continue to support select's native behavior and continue to use the select's dropdown UI. This minimal approach will attempt to wrap the select element with a div.select.

By wrapping it, we'll be able to style .select; but we'll need to remove all the native UI styling of select.
