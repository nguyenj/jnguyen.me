Styling checkboxes is super simple. With the basic markup, CSS is all that is needed to pull off some awesome creative checkboxes without the use of JavaScript!

Normally, I start off with the basic markup:

[code lang=text]
<fieldset>
<input type="checkbox" id="check-me" />
<label for="check-me">Check Me</label>
</fieldset>
[/code]

Placing those elements inside of a fieldset just helps organize the form markup just a bit. In order fully customize the checkboxes, this order cannot be compromised (input then label)â€“I'll explain why a little later. The for attribute on the label element allows the user to click on the label which will trigger the checkbox input with the same id value of the for attribute; in other words, this label is for the id value of check-me.
